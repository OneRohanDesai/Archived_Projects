# **NyxCloud**

NyxCloud was an experimental platform designed to make Infrastructure as Code (IaC) deployment dramatically easier.
It allowed users to **browse**, **deploy**, and **manage** AWS automation scripts with a streamlined interface and one-click workflows.

This repository serves as an archival snapshot of the project and includes two demo videos showcasing the platform.

---

## 🚀 **Features (from original platform)**

* Browse curated AWS IaC automation scripts
* One-click deploy, monitor, and destroy workflows
* Simple dashboard for tracking infrastructure state
* Admin interface for managing templates and user operations
